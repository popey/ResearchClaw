# Paper Summarization

Use this skill when the user wants a concise or structured summary of a research paper.

## Tools

- `summarize_paper`: summarize a paper from a file path, URL, or arXiv ID

## Guidance

- Use `brief` for a quick preview.
- Use `standard` for a normal structured summary.
- Use `detailed` only when the user explicitly wants section-level detail or tables.
