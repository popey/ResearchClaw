# Research Notes

Use this skill when the user wants to save observations, tag ideas, or search prior notes.

## Tools

- `create_note`: create a structured note with tags and optional paper references
- `search_notes`: search note content and titles
- `list_note_tags`: inspect existing note tags

## Guidance

- Prefer clear, reusable titles.
- Encourage tags when the user wants future retrieval.
- Use `search_notes` before creating a duplicate note on the same topic.
